# DriveFleet

Welcome to **DriveFleet** — a modern, responsive car rental marketplace built on the client side to help users discover, book, and manage vehicles with speed and confidence.

## 🌐 Live Site

Experience the live client app directly at:

**https://drivefleet-client.vercel.app**

> This URL points to the DriveFleet client-side experience for browsing cars, managing bookings, and accessing the full rental workflow.

## 🚗 What DriveFleet Offers

DriveFleet is designed to deliver a polished rental experience with a clean user interface and smart booking features.

- Fast client-side browsing of available cars with responsive filters and search.
- Seamless booking flow for customers to reserve vehicles in a few clicks.
- Secure authentication and account management for registered users.
- Engaging car detail pages with rich vehicle features and high-quality visuals.
- Admin-friendly car management and booking review from the client dashboard.

## ✨ Why Choose DriveFleet

DriveFleet blends modern web design, performance, and a customer-first rental journey. It is perfect for startups, agencies, or fleet operators who want a fresh vehicle booking platform that feels intuitive and reliable.

## 📦 Project Structure

The app is built with Next.js and organized for clarity on the client side:

- `app/` — main Next.js application routes and pages
- `src/component/` — reusable UI components like navigation, cards, and modals
- `src/lib/` — client-side helpers for authentication and API access
- `public/` — static assets and media for the UI

## 🚀 Getting Started Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to explore the DriveFleet client app locally.
